### Get started

create a python virtual environment
```bash
python -m venv .venv
```
activate the venv (linux)
```bash
source .venv/bin/activate
```
activate the venv (windows)

```powershell
.\venv\Scripts\activate
```
install the dependecies
```bash
pip install -r requirements.txt
```
